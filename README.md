## Setup

```bash
git clone https://github.com/your-user/cookbox.git
cd cookbox
pnpm install
pnpm dev
```

---

## Config

Create a .env file with your API key:

```bash
VITE_HF_ACCESS_TOKEN=your_api_key_here
```

- React
